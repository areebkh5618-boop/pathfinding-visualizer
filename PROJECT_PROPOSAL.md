## Introduction

Artificial Intelligence is becoming an important part of modern technology. Applications such as GPS navigation, robotics, games, and smart systems use path finding algorithms to find the shortest and most efficient routes. This project focuses on developing an interactive path finding visualizer that demonstrates how different AI search algorithms work in real-time. The system will help students understand algorithm behavior through graphical visualization and animations.

---

## Problem Statement

Many students face difficulty understanding path finding algorithms because most concepts are taught theoretically. Without proper visualization, it is hard to understand how algorithms explore nodes, avoid obstacles, and determine the shortest path. Therefore, an interactive educational tool is needed to improve learning and practical understanding of AI search techniques.

---

## Objectives

The main objective of this project is to create an interactive AI-based path finding visualizer using modern web technologies. The project will implement algorithms such as:

1. **Breadth First Search (BFS)**
2. **Depth First Search (DFS)**
3. **Dijkstra's Algorithm**
4. **A* Algorithm**

It will also include interactive map integration using Leaflet.js to simulate real-world path finding scenarios.

---

## Project Methodology

The project will be developed using the following approach:

### Frontend Development
- **React.js** for building interactive UI components
- **Tailwind CSS** for responsive and modern styling
- **JavaScript (ES6+)** for algorithm implementation

### User Interface
- A grid-based interface where users can:
  - Place starting points (🟢)
  - Place destination points (🔴)
  - Add obstacles (⬛)
  - Choose an algorithm to visualize

### Algorithm Visualization
- Step-by-step animation of algorithm execution
- Color-coded visualization of:
  - Visited cells (🔵)
  - Shortest path (🟡)
  - Obstacles (⬛)
- Real-time statistics and information

### Real-World Integration
- Leaflet.js integration for displaying routes on actual maps
- Ability to switch between grid view and map view
